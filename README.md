# Plataforma de Interoperabilidad Clínica: Pipeline ETL (No-OMOP a OMOP CDM) y Fachada HL7 FHIR

Este repositorio contiene el código fuente y la arquitectura de microservicios desarrollada para la transformación, normalización e intercambio estandarizado de datos clínicos. El proyecto resuelve la fragmentación de sistemas de salud legados mediante una arquitectura en dos fases: un proceso asíncrono de Extracción, Transformación y Carga (ETL) hacia el modelo analítico **OMOP CDM** (Common Data Model) y la posterior exposición de dichos datos en tiempo real bajo el estándar internacional **HL7 FHIR** utilizando el framework **HAPI FHIR**.
